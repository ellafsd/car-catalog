# Car Rental Catalog
A car rental catalog application built with React and TypeScript.  
Users can filter vehicles by brand, model, and year, with pagination and URL-based state management.


## Demo

https://github.com/user-attachments/assets/c09b1aeb-3fcd-480e-be60-833d1cf88074



## Tech Stack
- React
- TypeScript
- Vite
- React Router DOM
- React Select
- React Paginate
- Motion
- Tailwind CSS
- Axios

## Sources
Car data is provided by the following public API:  
https://public.opendatasoft.com/explore/dataset/all-vehicles-model/information/?sort=year
